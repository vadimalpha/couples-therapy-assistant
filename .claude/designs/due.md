# Due Analysis

## Overview
- **App**: Due - Reminders & Timers
- **Type**: Inspiration (reminder and notification design)
- **Analyzed**: 2025-08-05
- **Relevance**: Persistent reminders, clean notification design, time-based organization
- **Platform**: iOS app

## Visual Design
### Color Palette
- Primary: #000000 (black UI on dark mode shown)
- Accent: #FF3B30 (red for overdue/urgent)
- Secondary: #007AFF (blue for active timers)
- Background: Pure black in dark mode
- Text: #FFFFFF (white on dark)

### Typography
- Monospace for time displays
- Clean sans-serif for reminder text
- Large, readable time formats
- Clear hierarchy between time and description

### Visual Style
- Ultra-minimalist
- Focus on function over form
- High contrast for readability
- No decorative elements
- Time-centric design

## App Structure
### Navigation
- List-based interface
- Sections for different time periods
- Simple toolbar at bottom
- Minimal navigation complexity

### Reminder Organization
1. **Time-Based Grouping**
   - Overdue (in red)
   - Today
   - Tomorrow
   - This Week
   - Future dates

2. **Reminder Properties**
   - Time display prominent
   - Description below
   - Repeat options
   - Alert customization

3. **Quick Actions**
   - Snooze options readily available
   - Mark complete
   - Reschedule easily

## User Experience
### Persistent Notifications
- Key feature: Reminders repeat until acknowledged
- Customizable snooze intervals
- Multiple alert options
- Auto-snooze functionality

### Time Input
- Natural language processing
- Quick time presets
- Relative time options (in 1 hour, etc.)
- Flexible scheduling

### Visual Feedback
- Color coding for urgency
- Clear overdue indicators
- Animation on completion
- Sound/haptic feedback

## Key Takeaways for Terminus
1. **Persistent Reminders**: Implement "nag" functionality for important follow-ups
2. **Time-Centric Display**: Make next action time prominent
3. **Quick Snooze**: Easy postponement without losing the reminder
4. **Natural Input**: Accept flexible time inputs ("next week", "in 2 days")
5. **Urgency Indicators**: Use color to show overdue connections
6. **Minimal Interface**: Focus on the essential information
7. **Smart Defaults**: Suggest follow-up times based on relationship type
8. **Batch Actions**: Handle multiple reminders at once
