# Things 3 Analysis

## Overview
- **App**: Things 3
- **Type**: Inspiration (task and activity tracking)
- **Analyzed**: 2025-08-05
- **Relevance**: Excellent UI for organizing activities, clean iOS design, smart organization
- **Platform**: iOS app

## Visual Design
### Color Palette
- Primary: #007AFF (iOS system blue for interactive elements)
- Accent: #FF3B30 (red for important/overdue items)
- Secondary: Various categorical colors (teal, purple, green, orange)
- Background: #FFFFFF (pure white)
- Text: #000000 (primary), #8E8E93 (secondary gray)

### Typography
- Clean system font (SF Pro)
- Strong hierarchy with size and weight variations
- Generous line height for readability
- Monospace for dates/times

### Visual Style
- Minimalist and refined
- Extensive use of white space
- Subtle animations and transitions
- No unnecessary visual elements
- Focus on content over decoration

## App Structure
### Navigation
- Clear sections: Inbox, Today, Upcoming, Anytime, Someday
- List-based organization with nested projects
- Calendar integration for time-based views

### Organization Patterns
1. **Smart Lists**
   - Today: Current focus items
   - Upcoming: Scheduled future items
   - Anytime: Unscheduled but active
   - Someday: Ideas for later

2. **Projects & Areas**
   - Projects: Goal-oriented groupings
   - Areas: Ongoing responsibilities
   - Tags: Cross-cutting categorization

3. **Task Properties**
   - Notes for context
   - Tags for categorization
   - Deadlines with smart notifications
   - Checklists within tasks

## User Experience
### Quick Capture
- Plus button always accessible
- Natural language input
- Quick scheduling options
- Minimal friction to add items

### Visual Feedback
- Checkmark animations on completion
- Progress rings for projects
- Color coding for urgency
- Clear state indicators

### Gesture Support
- Swipe actions for quick operations
- Drag and drop for reorganization
- Pull to refresh
- Long press for context menus

## Key Takeaways for Terminus
1. **Clean Lists**: Use simple, scannable lists for activities/interests
2. **Smart Organization**: Implement automatic grouping (upcoming connections, overdue follow-ups)
3. **Quick Add**: Floating action button for rapid interest/activity capture
4. **Visual Progress**: Show relationship "health" or interaction frequency visually
5. **Natural Language**: Allow flexible input for activities and reminders
6. **Contextual Actions**: Swipe gestures for common actions (remind, archive, delete)
7. **Time-Based Views**: Show contacts by last interaction or upcoming reminders
8. **Minimal Chrome**: Let content shine with minimal UI decoration
